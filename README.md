# techpunch-clj

Common clojure code that can be reused across projects.


## Usage

Latest tag: ![](https://img.shields.io/github/v/tag/techpunch/techpunch-clj), and sha: ![](https://img.shields.io/github/sha/techpunch/techpunch-clj)

In deps.edn:

```edn
{techpunch/techpunch-clj {:git/url "https://github.com/techpunch/techpunch-clj.git"
                          :git/tag "(latest tag)"
                          :git/sha "(latest sha)"}}
```
